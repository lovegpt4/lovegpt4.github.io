【绝对教会】国内ChatGPT账号注册教程，无需手机验证码（2024年1月更新）

Posted on 2023-12-12 | In chatgpt |
本文解释了如何在中国内地注册和使用ChatGPT。文章首先建议使用海外邮箱进行注册，特别推荐Proton邮箱。接着，详细描述了如何创建Proton邮箱账户，包括免费套餐的选择、设置用户名和密码，以及如何通过真人检验而不需要手机验证。文章还涵盖了如何在OpenAI上注册账号，包括验证邮箱、填写个人信息和选择账户类型。最后，文章介绍了如何购买GPT Plus会员，以及如何为API开发者用户获取GPT-4的使用权限。

之前注册 ChatGPT 除了有一个海外邮箱，还需要手机号接码，一般都是推荐一些虚拟手机号接码平台，比如 sms-activate.org

但是现在通通不用！

只需要你能科学上网，即可注册。

（如果还不会可以用这个：科学稳定访问 openai

注册完后，或者你已经有账户，想要快速升级 Plus，可以看这篇教程：ChatGPT 2分钟升级 Plus方法

国内的邮箱都不再允许注册 ChatGPT，建议申请一个免费的海外邮箱 Proton 来注册，其他海外邮箱，如 Gmail 也可以，但是注册难度高一些，步骤繁琐。

一、注册 proton 邮箱

1.1 访问 proton

https://proton.me/

选择右上角的 「Create a free account」

1.2 选择免费套餐

1.3 选择邮箱后缀，设置用户名和密码

推荐选择 proton.me

1.4 通过真人检验，跳过手机验证

依次拖动验证码，然后设置显示的名称，跳过手机验证就可以。

注册完后，可以去登陆邮箱，方便接收验证码：

https://proton.me/

二、注册 OpenAI 账号

2.1 申请 openai

首先打开 OpenAI 的账户注册页面，输入邮箱:

https://chat.openai.com/

首先验证邮箱，去 Proton.me 收件箱里点击「Verify email address」即可。

然后提供你的名和姓及生日，并点击继续

然后选择您创建 API 还是 ChatGPT账户即可。

个人用户一般都选择 ChatGPT 账户即可：

Openai 的用户体系分成两大类：ChatGPT 用户和API开发者用户，两个体系是相互独立的，用户名密码是一致的，但是访问入口和支付都是独立的：

ChatGPT 用户的登录入口是 https://chat.openai.com
API 开发者用户的登录入口是 https://platform.openai.com

ChatGPT 用户获取 GPT-4 的使用权限，是通过购买 GPT Plus 会员，会员的收费标准是 20 美元/每月。

购买 GPT Plus 会员后，可以在 Chat 界面选择 GPT4 进行聊天。

购买教程在这里:

ChatGPT 2分钟升级 Plus方法

API 开发者用户获取 GPT-4 API的使用权限，是通过升级付费帐户，升级付费帐户的教程在这里。

然后需要预充值5美元以上，才能开通GPT-4的访问权限
