1月26号凌晨，OpenAI 宣布推出5种新模型，包括两款新的嵌入式模型、更新后的 GPT-4 Turbo 预览模型、GPT-3.5 Turbo 模型以及文本审核模型。

此外，chatgpt的价格也有低至50%的下降。新模型的输入价格下降 50% 至 $0.0005 /1K tokens，输出价格下降 25% 至 $0.0015 /1K tokens。

OpenAI 还发布了更新后的 GPT-4 Turbo 预览模型 gpt-4-0125-preview，旨在减少模型的“懒惰”情况。

1. 五种新模型

1.1 两款嵌入式模型

OpenAI 引入了两款新的嵌入式模型：一个体积更小、效率更高的 text-embedding-3-small 模型，以及一个更大、性能更强的 text-embedding-3-large 模型。

1.1.1 新小型文本嵌入模型：text-embedding-3-small

text-embedding-3-small 是一款高效的新嵌入式模型，与其前身 text-embedding-ada-002 相比，在多语言检索常用基准（MIRACL）上的平均得分从 31.4% 提高到 44.0%，在英语任务常用基准（MTEB）上的平均得分从 61.0% 提高到 62.3%。

新模型和之前模型的性能比较。

Eval benchmark ada v2 text-embedding-3-small text-embedding-3-large
MIRACL average 31.4 44.0 54.9
MTEB average 61.0 62.3 64.6

此外，与 text-embedding-ada-002 相比，text-embedding-3-small 的价格降低了 5 倍，每 1000 个 token 的价格从 0.0001 美元降至 0.00002 美元。

1.1.2 新大型文本嵌入模型：text-embedding-3-large

text-embedding-3-large 是新一代的大型嵌入式模型，可以创建多达 3072 维的嵌入。在 MIRACL 上的平均得分从 31.4% 提高到 54.9%，在 MTEB 上的平均得分从 61.0% 提高到 64.6%。

根据openai，text-embedding-3-large定价是$0.00013 / 1k tokens.

1.2 GPT-3.5 Turbo 新模型

下周，OpenAI 将推出新的 GPT-3.5 Turbo 模型 gpt-3.5-turbo-0125，并第三次在过去一年内降低 GPT-3.5 Turbo 的价格，以帮助客户扩展。新模型的输入价格下降 50% 至 $0.0005 /1K tokens，输出价格下降 25% 至 $0.0015 /1K tokens。

1.3 GPT-4 Turbo 预览更新

自 GPT-4 Turbo 发布以来，超过 70% 的 GPT-4 API 客户已经转移到 GPT-4 Turbo。今天，OpenAI 发布了更新后的 GPT-4 Turbo 预览模型 gpt-4-0125-preview，旨在减少模型未完成任务的“懒惰”情况。

1.4 更新后的文本审核模型

作为 OpenAI 持续安全工作的一部分，发布了迄今为止最强大的文本审核模型 text-moderation-007。text-moderation-latest 和 text-moderation-stable 别名已更新为指向该模型。

2. 两个新功能

OpenAI 推出了两项改进措施，即API 使用情况和 API 密钥管理新方式，为开发者提供更多关于他们使用情况的可见性和对 API 密钥的控制。

首先，开发者现在可以从 API 密钥页面为 API 密钥分配权限。其次，使用仪表板和使用情况导出功能现在在开启追踪后可以展示 API 密钥级别的指标，简化按功能、团队、产品或项目级别查看使用情况。

OpenAI 将在未来几个月进一步改进开发者查看 API 使用情况和管理 API 密钥的能力，尤其是在大型组织中。
